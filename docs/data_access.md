**Data Access**  
All large raw data files (such as `data_raw/`) for this project are hosted on [OSF](https://osf.io/). The repository contains only placeholder folders for these large materials. To obtain the actual data:

1. Go to the [OSF project page](https://osf.io/).
2. Download the relevant data (e.g., `data_raw.zip`)
3. Replace the placeholder folder in your local copy of the repository with the downloaded data.

**Distribution Requirements**  
All shared datasets are fully de-identified and provided only for academic use. Redistribution of the dataset, in whole or in part, to third parties is not permitted without explicit permission from the research team. Please cite the project as indicated in the README if you use the data in publications or presentations.
